COURSES OFFERED :-

  1. CLASS ( 1st to 5th ) : All Subjects.
  2. CLASS ( 6th to 8th ) : Maths, Science, English and Social-Science.
  3. CLASS (9th and 10th ): Maths, Science and English.
  4. CLASS (11th and 12th): Physics, Chemistry and Maths.
  5. SPECIAL COURSES for Sanik School, R.K Mission, C.H.S(B.H.U), Navodaya, Simultala Awasiya Vidyalaya and other competitive school exams.

It is a premier institute for covering Academic Syllabus along with other competitive school exams.
